# Conversor de Moneda (Java)

Proyecto simple en Java para convertir entre monedas. Incluye lógica de conversión, interfaz (CLI o GUI según la implementación) y pruebas básicas.

## Requisitos
- JDK 21
- gson-2.13.2.jar
- (Opcional) IDE: IntelliJ IDEA, Eclipse, VS Code

## Cómo contribuir
1. Crear un issue describiendo el cambio o bug.
2. Hacer un fork y crear una rama feature/mi-cambio.
3. Añadir pruebas si aplica.
4. Hacer un pull request describiendo los cambios.
